## 📖 项目简介

本仓库包含两款独立的终端游戏，均使用 C++ 编写，支持跨平台运行：

| 游戏 | 类型 | 说明 |
|:---:|:---:|:---|
| **RPG** | 回合制 RPG | 操控角色使用普攻、技能、大招进行战斗，包含敌人系统、背包系统与关卡选择 |
| **终端音游** | 4K 下落式音游 | FGHJ 四轨道操作，支持谱面导入、存储与游玩 |

其中本仓库仍包含一款开发中的工厂游戏

## 🕹️ RPG — 回合制RPG

### 游戏特色

- ⚔️ **回合制战斗** — 普攻(A)、技能(E)、大招(Q) 三种操作方式
- 👾 **多样敌人** — 普通敌人 / 精英敌人 / Boss 三级分类，各具独特AI行为
- 🎒 **背包系统** — 战斗奖励收集与物品管理
- 🗺️ **关卡系统** — 多关卡选择，逐步挑战
- 📈 **养成系统** — 角色成长与升级（开发中）

### 操作说明

| 按键 | 操作 |
|:---:|:---|
| `a` | 普攻 — 对指定敌人造成基础伤害，积累理性点 |
| `e` | 技能 — 消耗理性点发动特殊技能 |
| `q` | 大招 — 消耗能量值释放强力技能 |

## 🎵 终端音游 — 4K下落式

### 游戏特色

- 🎹 **4轨道下落式** — 经典音游玩法，终端实现
- 📝 **自定义谱面** — 支持谱面导入、存储与删除
- ⚡ **精确判定** — 一帧（8ms）为基本判定单位

### 操作说明

| 按键 | 轨道 |
|:---:|:---:|
| `F` | 第1轨道 |
| `G` | 第2轨道 |
| `H` | 第3轨道 |
| `J` | 第4轨道 |

### 判定规则

| 判定 | 范围 |
|:---:|:---|
| **Good** | ±4帧（±100ms） |
| **Bad** | 4~12帧（100~300ms） |
| **Miss** | 超出判定范围 |

### 谱面制作

谱面为纯文本格式：

```
谱面名称
谱面时长
音符起始帧 音符结束帧 轨道号
音符起始帧 音符结束帧 轨道号
...
```

> 注意：同轨道内前后两个音符间距应大于12帧

## 🛠️ 编译与运行

### 环境要求

- C++ 编译器（支持 C++11 及以上）
- Windows / Linux / macOS

### 编译

```bash
# game.1 回合制RPG
cd RPG
g++ game.cpp -o game

# 终端音游
cd 终端音游
g++ main.cpp -o main
```
关于编译：理论上本项目内部自带项目的.exe文件和Unix执行文件，所以如果该文件没有问题可以直接使用


### 运行

```bash
# 回合制RPG
./game

# 终端音游（需将谱面放置在 staff 文件夹中）
./main
```
